查看WebService请求以及返回报文

解决思路

Webservice 本质上还是http的请求，所以通过抓包即可拿到对应的请求以及返回的报文。其中抓包工具使用 Charles，免费使用一个月。

具体办法

  1. 下载 Charles，设置代理的端口号并启用代理： proxy -> proxy settings -> enable transparent http proxying 选中
  2. 在调用 webservice前设置下代理，如下
System.setProperty("http.proxyHost", "192.168.1.1");
System.setProperty("http.proxyPort", "8888");
